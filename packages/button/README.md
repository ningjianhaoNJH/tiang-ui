# @tiang-ui/button

## Install

```bash
npm i @tiang-ui/button
# or
yarn add @tiang-ui/button
```

## Usage

```vue
<template>
 <tiang-button/>
</template>

<script lang="ts">
import { defineComponent, toRefs } from 'vue'
import TiangButton from '@sum-ui/button'

export default defineComponent({
  components: { TiangButton },
  setup() {}
})
</script>
```
